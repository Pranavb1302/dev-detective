# DevDetective

DevDetective is a web application that allows users to search and explore GitHub
profiles. It fetches public user data and repositories using GitHubâ€™s REST API
and displays them in a clean and simple interface.

## Features
- Search GitHub users by username
- Display profile details and repositories
- Real-time API-based data fetching
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- GitHub REST API

## Project Overview
The project demonstrates API consumption and dynamic rendering of data on the
frontend. It focuses on handling asynchronous requests and presenting structured
information to users.
